# 文件处理器

一个用于批量处理文件的桌面工具。工具通过 Excel 表格维护文件清单，并根据表格内容对指定文件夹中的文件进行读取、复制、移动、替换、改名和删除。

## 主要功能

- 文件读取：扫描文件夹中的文件，并把文件名写入 Excel 表格。
- 文件复制：按照 Excel 的文件名清单，从起始文件夹复制文件到目标文件夹。
- 文件移动：按照 Excel 的文件名清单，从起始文件夹移动文件到目标文件夹。
- 文件替换：按照文件名在目标文件夹及其子文件夹中查找同名文件，找到后用起始文件夹中的同名文件覆盖。
- 文件改名：按照 Excel 中的旧名称和新名称批量修改文件名。
- 文件删除：按照 Excel 的文件名清单删除起始文件夹中的文件。
- 文件夹改名：勾选“仅修改文件夹”后，可以按照 Excel 中的名称映射批量修改文件夹名。
- 包含子目录：勾选后，读取、复制、移动、替换、改名、删除会同时处理子文件夹中的内容。

## 界面配置说明

打开工具后，主界面会显示以下配置：

| 配置项 | 说明 |
| --- | --- |
| 文件起始地址 | 原始文件所在文件夹。复制、移动、删除、改名时会从这里查找文件；替换时这里放“新文件”。 |
| 文件目标地址 | 复制、移动、替换的目标文件夹。替换时会在这里及其子文件夹中查找同名文件并覆盖。 |
| 表格文件地址 | 用于保存或读取文件清单的 Excel 文件。 |
| 工作表名称 | Excel 中要使用的工作表名称。 |
| 文件名列 | 保存完整文件名的列，例如 `1` 表示 A 列。复制、移动、替换、删除会读取这一列。 |
| 原名称列 | 保存原文件名主体的列，主要用于普通改名。 |
| 新名称列 | 保存新文件名主体的列，主要用于普通改名。 |
| 写入列 | 保存新完整文件名的列，主要用于“全名称操作”。 |

点击“修改数据”可以修改这些配置。配置会保存到本机，下次打开工具时会自动读取。

## Excel 表格格式

表格第一行是标题，从第二行开始写文件数据。

默认列含义如下：

| A 列 | B 列 | C 列 | D 列 |
| --- | --- | --- | --- |
| 文件名列 | 原名称列 | 新名称列 | 写入列 |

示例：

| 文件名列 | 原名称列 | 新名称列 | 写入列 |
| --- | --- | --- | --- |
| 合同.pdf | 合同 | 合同-新版 | 合同-新版.pdf |
| 图片.jpg | 图片 | 图片-归档 | 图片-归档.jpg |

如果使用“文件读取”，工具会自动把扫描到的文件写入这些列。

## 操作说明

### 1. 初始化数据表格

点击“初始化数据表格”，工具会清空当前工作表，并写入标准标题：

- 文件名列
- 原名称列
- 新名称列
- 写入列

注意：初始化会清空整个工作表内容，请先确认表格中没有需要保留的数据。

### 2. 文件读取

选择“文件读取”，点击“执行”。

工具会扫描“文件起始地址”中的文件，并写入 Excel 表格。勾选“包含子目录”后，也会扫描子文件夹中的文件。

### 3. 文件复制

选择“文件复制”，点击“执行”。

工具会读取 Excel 的“文件名列”，从“文件起始地址”查找对应文件，并复制到“文件目标地址”。

如果勾选“包含子目录”，工具会在起始文件夹及子文件夹中查找文件，并尽量保留原来的相对目录结构。

### 4. 文件移动

选择“文件移动”，点击“执行”。

逻辑和“文件复制”类似，但处理完成后，原文件会从起始文件夹中移走。

### 5. 文件替换

选择“文件替换”，点击“执行”。

使用方式：

1. 把新文件放在“文件起始地址”中。
2. 把需要被替换的文件所在根目录设置为“文件目标地址”。
3. 在 Excel 的“文件名列”中填写要替换的文件名。
4. 执行后，工具会在目标文件夹及其子文件夹中遍历查找同名文件，找到后用起始文件夹中的同名文件覆盖。

示例：

- 起始地址中有：`报价单.xlsx`
- 目标地址及子文件夹中有多个同名的：`报价单.xlsx`
- 执行“文件替换”后，目标地址里所有同名的 `报价单.xlsx` 都会被起始地址中的文件覆盖。

注意：

- 如果起始地址里找到多个同名源文件，工具会停止并提示错误，避免误用错误文件替换。
- 替换会覆盖目标文件，请确认文件无误后再执行。

### 6. 文件改名

选择“文件改名”，点击“执行”。

普通“执行”使用：

- 原名称列 -> 新名称列

适合只修改文件名主体，并保留原扩展名。

点击“全名称操作”时使用：

- 文件名列 -> 写入列

适合直接把完整文件名改成另一个完整文件名，例如从 `合同.pdf` 改成 `合同-新版.docx`。

### 7. 文件夹改名

勾选“仅修改文件夹”，再选择“文件改名”并执行。

此时工具会改文件夹名，而不是改文件名。勾选“包含子目录”后，会包含子文件夹，并从层级较深的文件夹开始处理。

### 8. 文件删除

选择“文件删除”，点击“执行”。

工具会读取 Excel 的“文件名列”，并从“文件起始地址”删除匹配到的文件。勾选“包含子目录”后，会在子文件夹中一起查找。

删除操作不可恢复，请谨慎使用。

## 启动和打包

开发环境运行：

```bat
run.bat
```

生成 EXE：

```bat
build_exe.bat
```

打包完成后，EXE 文件会生成在 `dist` 文件夹中。

## 依赖

项目依赖：

- PySide6：桌面界面
- openpyxl：读写 Excel 表格
- PyInstaller：生成 EXE 时使用

依赖版本记录在 `requirements.txt` 中。

## 使用建议

- 批量替换、删除、移动前，建议先备份重要文件。
- 如果不同子文件夹里存在大量同名文件，请确认 Excel 清单和起始文件夹中的源文件是否唯一。
- 执行“文件读取”可以快速生成标准表格，再在 Excel 中修改新名称后执行改名。
